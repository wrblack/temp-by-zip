# Temperature by zip code site

## What you need:
1. node v4+
2. a server/web host that can run Node apps

## What you do:
1. `git clone` into directory of choice.
2. `cd` into directory.
3. `npm install` or better yet with Yarn, `yarn`

## Node modules:
Read `package.json` for what NPM/Yarn will get.
